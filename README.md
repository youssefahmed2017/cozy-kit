# cozy-kit

> A cozy Python package with greetings, quotes, bedtime stories, timers, and text utilities.

Created by **Youssef**

---
# Requirements

- Python +3.11

---

# Installation

## Install

```bash
pip install cozy-kit
```



## Upgrade

```bash
pip install --upgrade cozy-kit
```

---

# Quick Example
## Usage
```python
from cozy_kit import Greeting

user = Greeting(
    name="Youssef",
    nickname="Yoyo"
)

print(user.welcome())
```

## Output
```text
Welcome Youssef!
Or welcome Yoyo!
```
